# Styles 
```js
// src/styles/_variables.scss

$gold: #C5A059;       // The primary accent for icons and subtitles
$dark: #2D3134;       // The main text color for "JessaBach"
$light-bg: #F9F9F9;   // Background for sections like Testimonials
$white: #FFFFFF;      // Pure white for cards and high-contrast areas

$serif-font: 'Playfair Display', serif; // Recommended for that "Bach" feel
$sans-font: 'Inter', sans-serif;         // For clean "School of Music" text
```
